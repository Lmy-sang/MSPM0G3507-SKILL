 # 2026 电赛控制题核心元器件清单

 ## 运动平台（必选）

 | 元器件 | 备注 |
 |--------|------|
 | 四旋翼飞行器 | 最大轴距不超过 450mm |
 | 电动小车 | 长度≤35cm, 宽度≤25cm, 禁用履带和麦氏轮 |

 ## 主控/处理平台

 | 元器件 | 用途 |
 |--------|------|
 | 嵌入式开发系统板 | 主控制器（STM32 / MSP430 等） |
 | TI MSPM0 系统板 | TI 官方推荐控制平台（含板载调试下载） |
 | 微处理器最小系统板 | 备选 MCU 方案（含板载调试下载） |
 | 可编程逻辑器件及其下载板 | FPGA/CPLD，辅助高速逻辑/并行处理 |

 ## 电机与驱动

 | 元器件 | 用途 |
 |--------|------|
 | 小型电机及驱动器 | 小车驱动/云台/舵机等运动控制 |
 | 大功率器件 | MOS 管/IGBT 等，电机驱动的功率级 |

 ## 传感器

 | 元器件 | 用途 |
 |--------|------|
 | 光电传感器 | 循迹/避障/测速（对射/反射式红外等） |
 | 角度传感器 | 姿态检测/平衡控制（陀螺仪/加速度计/编码器等） |
 | 摄像头（可带处理器模块） | 视觉寻迹/目标识别（OpenMV/K210 等） |

 ## 通信与显示

 | 元器件 | 用途 |
 |--------|------|
 | 短距离无线通信模块 | 遥控/数传（蓝牙/WiFi/nRF24L01/LoRa 等） |
 | 无线图像传输模块 | 图传（FPV 第一视角/远程监控） |
 | 显示屏（≥6 吋） | 本地状态显示/调试界面 |

 ## 执行与输出

 | 元器件 | 用途 |
 |--------|------|
 | 小型继电器 | 开关型控制输出 |
 | 发光二极管 | 状态指示/信号灯 |
 | 蜂鸣器 | 声音提示/报警 |
 | 小型电磁铁 | 磁悬浮/吸附类控制任务 |
 | 7 段数码管 | 数值显示 |

 ## 信号调理与转换

 | 元器件 | 用途 |
 |--------|------|
 | A/D、D/A 转换器 | 传感器信号采集/模拟输出 |
 | 运算放大器、电压比较器、乘法器 | 信号调理/比较判断 |

 ## 可能涉及的控制任务结构件

 | 元器件 | 可能对应的控制场景 |
 |--------|---------------------|
 | 钢球（直径约 1cm） | 板球控制系统（ball-and-plate） |
 | 4 分 PPR 水管（外径约 20mm） | 液体/水位控制系统 |

 ## 非控制题专用（参考，大概率不需要准备）

 | 元器件 | 说明 |
 |--------|------|
 | 漆包线 | 绕制电感/变压器，偏电源题/信号题 |
 | 三相可变电阻器（50Ω/3A） | 负载类，偏电源题 |
 | 滑线变阻器（20Ω/5A） | 负载类，偏电源题 |
 | 单相自耦调压器（≥300VA） | 电源题专用 |
 | 单相隔离变压器（≥200VA） | 电源题专用 |
 | 磁环/磁芯 | 高频变压器/电感，偏电源题 |
 | 铁芯磁环（外径 80mm, 内径 50mm, 高 25mm） | 低频变压器，偏电源题 |
 | 镀锌薄铁片（厚度约 0.3mm） | 磁屏蔽/结构件，偏电源题 |
 | 可变增益集成宽带放大器 | 信号题用 |
 | DDS 芯片或模块 | 信号发生，偏信号题 |
