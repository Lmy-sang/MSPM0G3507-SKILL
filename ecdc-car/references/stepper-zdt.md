 # ZDT 张大头步进电机: Modbus RTU 闭环控制
 
 ZDT Emm42_V5.0 是总线型智能闭环步进驱动, 内置磁编码器和 FOC 闭环算法。通过 Modbus RTU 协议走 UART 控制。
 
 ## 硬件接线
 
 - 接口: UART TTL (直连 MCU, 无需 RS485 芯片)
 - 波特率: 默认 115200, 8N1
 - 电源: 12-24V (独立供电, 不能从 MCU 板取电)
 - MSPM0G: 接任意空闲 UART (S27F 底板舵机口是 PA8/TIMG0 PWM, 步进走 UART 需另选引脚)
 
 ## Modbus RTU 帧格式
 
 ```
 [从机地址(1B)] [功能码(1B)] [数据区] [CRC16_Lo(1B)] [CRC16_Hi(1B)]
 ```
 
 - 从机地址: 默认 `01H`
 - 功能码: `04H` 读, `06H` 写单寄存器, `10H` 写多寄存器
 - CRC16: 标准 Modbus CRC (多项式 0xA001)
 
 ## 关键指令速查表
 
 ### 控制指令
 
 | 操作 | 功能码 | 寄存器 | 帧数据 | 说明 |
 |------|--------|--------|--------|------|
 | 使能 | `10H` | `00 F3` | `01 10 00 F3 00 02 04 AB 01 00 00 CRC` | 使能状态填 1 |
 | 速度模式 | `10H` | `00 F6` | `01 10 00 F6 00 03 06 [dir] [acc] [rpm_hi] [rpm_lo] [sync] 00 CRC` | rpm 范围 0-3000 |
 | 位置模式 (相对) | `10H` | `00 FD` | `01 10 00 FD 00 05 0A [dir] [acc] [rpm] [pulse_hi] [pulse_lo] 00 [sync] CRC` | 模式=00 相对 |
 | 位置模式 (绝对) | `10H` | `00 FD` | 同上, 模式=01 绝对 | 绝对坐标, 原点为清零位置 |
 | 立即停止 | `10H` | `00 FE` | `01 10 00 FE 00 01 02 [sync] CRC` | |
 | 多机同步触发 | `10H` | `00 FF` | `01 10 00 FF 00 01 02 00 CRC` | |
 
 ### 读取指令
 
 | 操作 | 功能码 | 寄存器 | 帧数据 | 返回值 |
 |------|--------|--------|--------|--------|
 | 读转速 | `04H` | `00 35` | `01 04 00 35 00 02 CRC` | [符号, rpm] |
 | 读位置 | `04H` | `00 36` | `01 04 00 36 00 03 CRC` | [符号, 位置值] |
 | 读编码器 | `04H` | `00 31` | `01 04 00 31 00 01 CRC` | 编码器值 0-65535 |
 | 读状态 | `04H` | `00 3A` | `01 04 00 3A 00 01 CRC` | [使能/到位/堵转/保护] |
 
 ### PID 参数
 
 | 操作 | 寄存器 | 帧 |
 |------|--------|-----|
 | 修改 PID | `00 4A` | `01 10 00 4A 00 07 0E C3 [存] [Kp_32bit] [Ki_32bit] [Kd_32bit] CRC` |
 
 ### 辅助指令
 
 | 操作 | 寄存器 | 说明 |
 |------|--------|------|
 | 校准编码器 | `00 06` | `01 06 00 06 00 01 CRC` |
 | 位置清零 | `00 0A` | `01 06 00 0A 00 01 CRC` |
 | 解除堵转 | `00 0E` | `01 06 00 0E 00 01 CRC` |
 | 恢复出厂 | `00 0F` | `01 06 00 0F 00 01 CRC` |
 
 ## 字段说明
 
 - **dir**: 方向, `01`/`00` (某些版本 0=顺时针 CW, 1=逆时针 CCW)
 - **acc**: 加速度 0-255, 0=直接启动, 越大越平滑
 - **rpm**: 转速 0-3000 RPM
 - **pulse**: 位置脉冲数, 16细分下 3200 pulse = 1 圈
 - **sync**: 多机同步标志, 普通控制填 0
 - **位置角度换算**: `角度 = 位置值 × 360 / 65536`
 
 ## MSPM0G CRC16 实现
 
 ```c
 uint16_t Modbus_CRC16(uint8_t *data, uint16_t len) {
     uint16_t crc = 0xFFFF;
     for (uint16_t i = 0; i < len; i++) {
         crc ^= data[i];
         for (uint8_t j = 0; j < 8; j++) {
             if (crc & 0x0001) crc = (crc >> 1) ^ 0xA001;
             else crc >>= 1;
         }
     }
     return crc;  // 发送时先发低字节再发高字节
 }
 ```
 
 ## 典型控制流程
 
 ```
 上电 → 等待 2s → 发使能指令 → 确认使能成功 → 发速度/位置指令 → 循环读取状态
 ```
 
 **注意事项:**
 - 上电后等待驱动初始化 (约 2 秒), 再发指令
 - 首次使用需校准编码器 (`00 06`)
 - 步进电机独立供电, 不能从 C07A 的 5V/3.3V 取电
 - 位置模式下, 默认 16 细分, 3200 脉冲 = 1 圈
 - 如果堵转, 驱动会自动切断动力 (需发解除堵转指令恢复)
 
 ## 电机型号确认
 
 Emm42_V5.0 支持 28/35/42/57 四种规格电机。使用前通过上位机确认:
 - 电机类型设置 (1.8° 或 0.9°)
 - 通讯端口模式 (设置为 `2` = UART_FUN 串口通讯)
 - 通讯校验 (设置为 Modbus)
