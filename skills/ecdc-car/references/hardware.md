 # 硬件参考: MSPM0G3507 + C07A + S27F
 
 ## 平台总览
 
 | 组件 | 型号 | 备注 |
 |------|------|------|
 | MCU | TI MSPM0G3507 | Cortex-M0+, 80MHz, 128KB/32KB |
 | 核心板 | 轮趣 WHEELTEC C07A | V1.0 / V1.1 (V1.1 去掉 USER 按键, B8/B18/B19 引出) |
 | 底板 | S27F | S27F 与 S28A 内部走线相同, 仅丝印不同 |
 | 底盘 | 差速二驱 | TB6612 双路 PWM 驱动 |
 | 电机驱动模块 | 轮趣 D153C | TB6612FNG 双路驱动带稳压版, STBY 已接 3.3V |
 | 编码器 | 霍尔编码器 | 13线, 2倍频=26脉冲/转, 30:1减速比 |
 | 电源稳压 | RT9013-33GB | 5V → 3.3V LDO |
 | USB-UART | CH9102 | USART1, Type-C |

 ## 引脚互斥约束 (重要)
 
 以下外设共享引脚, 不可同时使用:
 
 | 约束 | 冲突外设 | 共享引脚 |
 |------|---------|---------|
 | 超声波 vs 雷达 | HC-SR04 ↔ 雷达模块 | PA24 |
 | 超声波 vs CCD巡线 | HC-SR04 ↔ CCD | PA9 |
 | CCD巡线 vs 红外巡线 | CCD ↔ 灰度模块 | PA27 |
 
 **比赛选题时, 先确定传感器组合, 再分配引脚, 避开冲突。**
 
 ### 禁用的固定引脚
 
 | 引脚 | 原因 |
 |------|------|
 | PA2~PA6 | 时钟引脚 (ROSC/LFXIN/HFXIN), 默认未焊接 |
 | PA19, PA20 | SWD 调试接口, 不可复用 |
 
 ## 编码器参数
 
 | 参数 | 值 | 说明 |
 |------|-----|------|
 | 编码器线数 | 13 | 每转 13 个脉冲 |
 | 倍频系数 | 2 | 只检测上升沿 |
 | 每转脉冲数 | 26 | 13 × 2 |
 | 减速比 | 30:1 | 电机轴:输出轴 |
 | RPM 公式 | `RPM = count × 60000 / (26 × sample_ms) / 30` | |
 
 ## TB6612 控制真值表
 
 | PWM | IN1 | IN2 | 功能 |
 |-----|-----|-----|------|
 | 1 | 0 | 1 | 正转 |
 | 1 | 1 | 0 | 反转 |
 | 1 | 0 | 0 | 制动停止 |
 | 1 | 1 | 1 | 制动停止 |
 | 0 | X | X | 停止 |
 
 **PWM 推荐频率: 10kHz, 范围 0-100kHz。STBY 已在模块上接 3.3V, 无需 MCU 控制。**
 
 ## 电源系统
 
 - 电机电源 VM: 7-12V (典型 12V, 经电池/电源输入)
 - 逻辑电源 VCC: 5V (模块稳压输出)
 - MCU 电源: 3.3V (板载 RT9013-33GB LDO)
 - 电池检测: 1/11 分压 (10k串联1k), ADC1 读取 PA15
 - 电池电压计算: `V_battery = ADC_value / 4096 * 3.3V * 11`
